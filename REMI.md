# GrantKeeper

GrantKeeper is a GitHub Pages-ready static mockup for a future nonprofit grant management application. It is intentionally built with plain HTML, CSS, and JavaScript so the concept can be reviewed without a build pipeline before being rebuilt in React or a similar framework.

## Mocked modules

- **Grant finder:** Searches sample funding opportunities and displays eligibility fit notes based on a nonprofit profile.
- **Expense tracker:** Tracks expenses across multiple grants with vendor/date/amount search, custom tags, and mocked invoice/proof-of-payment uploads.
- **Proposal studio:** Helps draft a proposal outline and links to grant writing tutorials plus bring-your-own-account AI tools such as ChatGPT and Claude.

## Running locally

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173`.
